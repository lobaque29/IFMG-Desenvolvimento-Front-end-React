Exemplo de Variáveis CSS

Variáveis CSS (ou propriedades customizadas) são como variáveis em linguagens de programação, permitindo armazenar valores (cores, fontes, tamanhos) em um só lugar e reutilizá-los no código para facilitar a manutenção e consistência, declaradas com --nome-da-variavel: valor; e usadas com var(--nome-da-variavel)
